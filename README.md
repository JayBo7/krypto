## Usage

> From the booking-module directory:
```sh
npm run build
npm start
```

## Requirements

An `nvmrc` file is included if using [nvm]

- Node 6.13.1

## Development
> For development:
Build the bundle.js
```sh
npm run dev
```
Run server
```sh
npm run start-dev
```

### Install Dependencies
```sh
npm install
```
